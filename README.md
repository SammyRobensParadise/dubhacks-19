# dubhacks-19
Application to allow people who are visually impaired to see text up close

# Setup:
1. `git clone`
2. `npm install`
3. `npm start`
4.  follow instructions for expo  CLI
5.  If you don't have `expo`installed run 
```npm install expo-cli --global```
